# ⏱ Обратный отсчёт

Приложение для контроля важных дат — сколько прошло и сколько осталось.

## 🚀 Установка через GitHub Pages

### Шаг 1 — Создайте репозиторий
1. Зайдите на [github.com](https://github.com) и войдите в аккаунт
2. Нажмите **New repository**
3. Назовите его `countdown` (или любое другое)
4. Выберите **Public**
5. Нажмите **Create repository**

### Шаг 2 — Загрузите файлы
1. В репозитории нажмите **Add file → Upload files**
2. Перетащите все файлы из этой папки:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon.png`
   - `README.md`
3. Нажмите **Commit changes**

### Шаг 3 — Включите GitHub Pages
1. Перейдите в **Settings → Pages**
2. В разделе **Source** выберите **Deploy from a branch**
3. Выберите ветку **main**, папку **/ (root)**
4. Нажмите **Save**

### Шаг 4 — Готово!
Через 1–2 минуты приложение будет доступно по адресу:
```
https://ВАШ_ЛОГИН.github.io/countdown/
```

## 📲 Установка как приложение

### На iPhone / iPad
1. Откройте ссылку в **Safari**
2. Нажмите кнопку **Поделиться** (квадрат со стрелкой)
3. Выберите **На экран «Домой»**
4. Нажмите **Добавить**

### На Android
1. Откройте ссылку в **Chrome**
2. Нажмите **⋮ → Установить приложение**
3. Нажмите **Установить**

### На компьютере (Chrome / Edge)
1. Откройте ссылку
2. В адресной строке нажмите иконку **установки** (➕ или монитор со стрелкой)
3. Нажмите **Установить**
